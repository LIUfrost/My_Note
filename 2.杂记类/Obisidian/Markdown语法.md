# 文本编辑
- 删除当前行：ctrl+shift+k
- 缩进：Tab
- 取消缩进：Shift+Tab
# 文本导航
- 将光标移到当前行的开头/结尾：Home/End
# 文本选择
- 取消选择：Esc
- 全选：ctrl+A
# 格式化文本
- 粗体：\*\*粗体文本\*\*
- 斜体：\*斜体文本\*
- 删除线：\-\-删除线文本\-\-
- 高亮：\=\=高亮文本\=\=
- 粗体+斜体：\*\*\*粗体和斜体文本\*\*\*
# 链接
- 内部链接：\[\[内部笔记\]\]
- 外部链接：\[标题\]\(内联链接\) （URL若包含空格，则用%20替换它们）
- 外部图片：\!\[Engelbart|100x145\]\(https://history-computer.com/ModernComputer/Basis/images/Engelbart.jpg\)  (若只指定了宽度：等比例缩放)
# 引用
- \> xxxxxxx
    \- xxxx
> xxxxxxx
   \- xxxx
# 列表
- \- xxxx
  \-xxxx
  \-xxxx
# 任务列表
- \- \[x\] 这是已完成的任务
- \- \[ \] 这是未完成的任务
- [x] 这是已完成的任务
- [ ] 这是未完成的任务
# 水平线
- 水平线： \-\-\-
# 代码
- 行内代码：\`代码\`
- 代码块：\`\`\`c
-               xxxxx
-               \`\`\`
# 表格
- 使用|和-来创建表格，数竖线用于分割列，段横线用于定义列标题。
